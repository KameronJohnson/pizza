Kam's Pizza Shoppe
==============

A JavaScript web app that allows a user to enter a desired pizza size and type, and  will return how many slices the pizza will yield based on the size.

This app uses JQuery, Mocha, Chai and Bootstrap.

Directions
-----

1) Open `pizza.html`

2) Have fun.
